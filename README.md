# Grokédex

**The Ultimate Grok Encyclopedia & Interactive Pokédex-Style Experience**

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Ready-blue?logo=github)](https://ryanjosephkamp.github.io/grokedex)

Grokédex is the comprehensive, living "Pokédex for Grok" — cataloging every major feature, capability, tool, connector, workflow, tip, and hidden gem of Grok and the Grok app.

Each entry is like a Pokémon entry: rich description, stats (complexity, power level), evolution paths, related entries, example prompts, pro tips, and original Grok-generated insights.

> *"Gotta know 'em all."*

## 🎯 Vision

- **Short-term**: High-quality GitHub-based blog & knowledge base with 12–20 in-depth posts written in Grok’s authentic voice.
- **Medium-term**: Beautiful interactive web app (searchable cards, filters, cross-references, progress tracking).
- **Long-term**: Living ecosystem with autonomous maintenance agents, structured data backend, and deep Grok integration.

## 📁 Repository Structure

- `GROKDEX_PROJECT_PLAN.md` — Full project plan and roadmap
- `planning/` — Skills cheat sheet and planning docs
- `posts/` — Blog posts / encyclopedia entries (Markdown)
- `templates/` — Entry templates and frontmatter
- `assets/` — Images and media
- `drafts/` — Work-in-progress content

## 🚀 Getting Started

This repo powers the Grokédex blog (via GitHub Pages or static site generator).

Content is created collaboratively with Grok using custom agents and published directly via GitHub tools.

See the [full project plan](GROKDEX_PROJECT_PLAN.md) for details on workflow, custom agents, autonomous updates, and the future interactive app.

## 🔄 Autonomous Maintenance

Future GitHub Actions + Grok-powered agents will keep content fresh with new Grok features and updates via draft PRs.
